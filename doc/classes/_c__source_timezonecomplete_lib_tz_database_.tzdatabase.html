normalizeLocal

  • normalizeLocal(zoneName: string, localTime: TimeStruct): TimeStruct
  • normalizeLocal(zoneName: string, localTime: number): number
    • Defined in c:/Source/timezonecomplete/lib/tz-database.ts:646

    Normalizes non-existing local times by adding a forward offset change. During a forward standard offset change or DST offset change, some amount of local time is skipped. Therefore, this amount of local time does not exist. This function adds the amount of forward change to any non-existing time. After all, this is probably what the user meant.

    Parameters

    • zoneName: string

      IANA time zone name

    • localTime: TimeStruct

      A local time, either as a TimeStruct or as a unix millisecond value

    Returns TimeStruct

  • totalOffsetLocal(zoneName: string, localMillis: number): Duration
    • Defined in c:/Source/timezonecomplete/lib/tz-database.ts:751

    Returns the total time zone offset from UTC, including DST, at the given LOCAL timestamp. Non-existing local time is normalized out. There can be multiple UTC times and therefore multiple offsets for a local time namely during a backward DST change. This returns the FIRST such offset. Throws if zone info not found.
